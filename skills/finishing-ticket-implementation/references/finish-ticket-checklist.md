# Finish Ticket Checklist

Use this checklist when closing one implemented ticket.

```markdown
## Ticket Close Checklist: <topic>/<N>

- [ ] Ticket status is `Implemented`
- [ ] Branch exists: `feat/<topic>-tN`
- [ ] Integration branch exists: `feat/<topic>-integration`
- [ ] Working tree is clean before checkout/merge
- [ ] Acceptance criteria evidence collected
- [ ] Validation method evidence collected
- [ ] Merge completed (or conflict reported)
- [ ] Post-merge tests passed
- [ ] Status moved to `Integrated`
- [ ] Status moved to `Closed`
- [ ] Completion log entry appended
```
